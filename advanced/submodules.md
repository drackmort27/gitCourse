# 📦 Git Submodules

## 🧠 ¿Qué son?

Repos dentro de otros repos.

---

## ➕ Agregar

git submodule add <repo>

---

## 📥 Clonar con submodules

git clone --recurse-submodules <repo>

---

## 🔄 Actualizar

git submodule update --init --recursive

---

## ⚠️ Uso

- Dependencias externas
- Librerías compartidas
