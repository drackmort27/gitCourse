# 🧠 Cheat Sheet

## Init

git init
git clone <url>

## Estado

git status
git log
git diff

## Staging

git add .
git add file.txt
git reset file.txt

## Commit

git commit -m "mensaje"

## Branch

git branch
git checkout -b rama

## Merge

git merge rama

## Remoto

git push
git pull
git fetch

## Historial

git show <hash>
git reflog

## Reset / Revert

git reset --soft <hash>
git reset --hard <hash>
git revert <hash>

## Otros

git stash
git stash pop
git tag
git clean
